# Владимирский АСК ДОСААФ России — Прыжки с парашютом

Официальный сайт Владимирского аэроклуба (АСК ДОСААФ России). Проект представляет собой современный, быстрый и защищенный веб-ресурс, построенный на чистом стеке PHP и Vanilla JS без тяжелых внешних фреймворков.

---

## 🛠 Технологический стек
- **Backend:** PHP 7.4+ (Архитектура без внешних зависимостей, PDO)
- **Database:** MySQL / MariaDB (с поддержкой транзакций и подготовленных запросов)
- **Frontend:** HTML5 (Семантический), CSS3 (Modern Glassmorphism, Flexbox/Grid), JavaScript ES6+ (Vanilla JS)
- **Server:** XAMPP (Apache) / Любой стандартный LAMP/WAMP/LEMP стек

---

## 📂 Структура проекта
- `index.php`, `about.php`, `prices.php`, `contacts.php`, `stocks.php` и др. — Основные публичные страницы сайта.
- `sky-control/` — Скрытая панель управления сайтом (Админка).
- `includes/` — Модульные компоненты (Header, Footer, навигация) и конфигурация подключения к БД (`config.php`, `db_connect.php`).
- `css/` — Таблицы стилей (глобальные стили `global.css`, модульные стили страниц и админки).
- `js/` — Клиентские скрипты (интерактивность, защита от DDoS, метео-виджет).
- `images/` — Медиа-активы (фотографии, иконки, логотипы клуба).
- `sql/` — Дампы базы данных (`init_db.sql`) со структурой таблиц и начальными данными.
- `docs/` — Дополнительная документация и инструкции по проекту.

## 🚀 Запуск проекта (Локально)

1. **Установка сервера:** Установите и запустите локальный сервер **XAMPP** (или OpenServer / MAMP / WAMP).
2. **Размещение файлов:** Скопируйте папку проекта в директорию веб-сервера (для XAMPP: `C:\xampp\htdocs\Sky`).
3. **Запуск служб:** В панели управления XAMPP запустите модули **Apache** и **MySQL**.
4. **Импорт базы данных:**
   - Откройте `http://localhost/phpmyadmin/`.
   - Создайте базу данных (по умолчанию `sky_db` или согласно вашим настройкам).
   - Выберите созданную базу данных, перейдите во вкладку **Импорт** и загрузите файл `sql/init_db.sql`.
5. **Настройка подключения:**
   - Проверьте и при необходимости отредактируйте параметры подключения к БД в файлах `includes/db_connect.php` и `includes/config.php` (имя пользователя, пароль, название базы данных).
6. **Открытие сайта:**
   - Перейдите в браузере по адресу: `http://localhost/Sky/`
   - Также в корне проекта доступен скрипт `ЗАПУСТИТЬ_САЙТ.bat` для быстрого автоматического открытия.

## 🔐 Вход в Админ-панель (sky-control)

В проекте реализована **многоуровневая защита административной панели** от несанкционированного доступа, сканирования директорий и подбора паролей (Brute-force).

### 1. Секретный пропуск (Защита от сканеров)
Если попытаться зайти в админ-панель напрямую по адресу `http://localhost/Sky/sky-control/` или `http://localhost/Sky/sky-control/login.php`, сервер вернет **фейковую ошибку 404 (Not Found)**. Это сделано для того, чтобы злоумышленники и боты не знали о существовании панели управления.

Чтобы открыть форму входа, необходимо использовать **секретный GET-параметр**:

👉 **Ссылка для входа:** `http://localhost/Sky/sky-control/login.php?entry=Semazino`

> **Примечание:** После перехода по этой ссылке система автоматически запомнит ваш пропуск в сессии и сделает чистый редирект на `login.php`, скрыв секретный параметр из адресной строки браузера.

### 2. Данные для авторизации (Логин и Пароль)
На открывшейся странице входа введите следующие учетные данные:

- **Логин:** `admin`
- **Пароль:** `sky2024`

### 3. Защита от подбора паролей (Brute-force)
При вводе неверного пароля срабатывает искусственная задержка ответа сервера (`sleep(2)`). Это делает автоматический перебор паролей по словарю неэффективным и практически невозможным.

### 4. Смена пароля администратора
Для изменения пароля администратора на продакшн-сервере:
1. Сгенерируйте новый хэш пароля с помощью PHP-функции `password_hash('ВАШ_НОВЫЙ_ПАРОЛЬ', PASSWORD_DEFAULT)`.
2. Откройте файл `sky-control/login.php`.
3. Замените значение константы `ADMIN_PASS_HASH` (строка 32) на полученный хэш.

## 🛡 Системы защиты и архитектурные особенности

### 🛑 Трёхуровневая защита от DDoS и парсеров
- **Rate Limiter (Ограничение запросов):** Автоматическое отслеживание активности сессии. При превышении лимита запросов в секунду IP временно блокируется на 10 секунд.
- **JS-Challenge:** Интеллектуальная защита, проверяющая способность браузера выполнять JavaScript (отсеивает ботов и скрипты автоматического парсинга). При подозрении на флуд пользователю показывается страница верификации, которая автоматически перенаправляет обратно после успешной проверки.
- **Фильтрация User-Agent:** Блокировка запросов с пустыми или подозрительными заголовками `User-Agent`.

### 🌤 Метео-индикатор летной годности
На сайте работает уникальный интерактивный виджет, который в реальном времени связывается с API OpenWeatherMap. Он получает актуальные погодные данные для аэродрома Семязино (скорость ветра, облачность, видимость) и выводит анимированный статус летной погоды с рекомендациями для парашютистов.

### ⚡ Производительность и оптимизация
- **База данных:** Использование подготовленных запросов PDO и глобального режима `PDO::FETCH_ASSOC` для обеспечения максимальной скорости и защиты от SQL-инъекций.
- **Оптимизация ассетов:** Асинхронная загрузка скриптов (`defer`), модульная структура CSS без дублирования кода, оптимизированное отображение изображений (`object-fit: cover`).
