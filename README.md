# Владимирский АСК ДОСААФ России - Новый стек (Vue + Fastify)

Проект Владимирского АСК ДОСААФ России (Прыжки с парашютом) был успешно мигрирован со старого монолитного PHP стека на современную, компонентную и быструю архитектуру. Теперь это современное SPA (Single Page Application) с отдельным API бэкендом.

## 🛠 Технологический стек

- **Фронтенд:** Vue.js 3 (Composition API), Vite, Vue Router, Axios. 
- **Бэкенд (API):** Node.js, Fastify.
- **База Данных (ORM):** MySQL, Prisma.
- **Админка:** Сохранена совместимость с PHP-панелью для управления заказами и контентом.

## ✨ Что было сделано (Ключевые изменения)

- **Полная миграция на Vue 3**: Все страницы (Цены, Команда, Отзывы, Акции и т.д.) переписаны на компоненты. Больше никаких перезагрузок страниц!
- **Fastify API**: Создан быстрый сервер для обработки заявок, новостей и акций.
- **Интеграция с БД (Prisma)**: Все данные теперь управляются через ORM, что делает работу с MySQL безопасной и структурированной.
- **Форма заказа сертификатов**: Добавлена строгая маска телефона `+7 (XXX) XXX-XX-XX` и валидация перед отправкой.
- **Исправление Отзывов**: Проведен полный аудит и сопоставление имен авторов с их фотографиями. 
- **Улучшенный Lightbox**: Фотографии в отзывах и благодарностях теперь открываются в полноэкранном режиме с качественным масштабированием.
- **Совместимость Админки**: Старая PHP админ-панель адаптирована под новую структуру базы данных (совместимость с PHP 7.4+ и новыми именами колонок).

## 📂 Структура проекта

```
/backend          - Модуль API (Fastify + Prisma)
  /prisma           - Схема базы данных
  /routes           - Эндпоинты API (/api/orders, /api/news и др.)
    
/frontend         - Клиентское приложение (Vue 3 + Vite)
  /src/components   - Общие компоненты (Header, Footer, Lightbox)
  /src/views        - Страницы (SPA маршруты)
  /public/images    - Медиа-активы
```

## 🚀 Установка и запуск

### Шаг 1: База данных (MySQL)
Запустите MySQL (например, через XAMPP) на порту `3306`. Убедитесь, что база `ask_dosaaf_db` создана.

### Шаг 2: Настройка окружения
В папке `backend/.env` укажите параметры:
```env
DATABASE_URL="mysql://root:@localhost:3306/ask_dosaaf_db"
PORT=3000
```

### Шаг 3: Запуск Бэкенда (из корня)
```bash
npm run dev_backend
```
Сервер запустится на `http://localhost:3000`.

### Шаг 4: Запуск Фронтенда (из корня)
```bash
npm run dev_frontend
```
Сайт будет доступен на `http://localhost:5173`.
