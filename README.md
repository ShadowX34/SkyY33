# Владимирский АСК ДОСААФ России - Новый стек (Vue + Fastify)

Проект Владимирского АСК ДОСААФ России (Прыжки с парашютом) был успешно мигрирован со старого монолитного PHP стека на современную, компонентную и быструю архитектуру.

## 🛠 Технологический стек

- **Бэкенд (API):** Node.js, Fastify, TypeScript
- **База Данных (ORM):** MySQL, Prisma
- **Фронтенд:** Vue.js 3, Vite, Vue Router, Vue Composition API
- **Аутентификация:** JWT

## 📂 Структура проекта

Оригинальные корневые `*.php` и `css/` файлы всё ещё присутствуют для обратной совместимости или истории, но главная рабочая среда теперь:

```
/backend          - Модуль API
  /src              - Точка входа в сервер
    /routes           - Обработчики эндпоинтов 
  /prisma           - Конфигурация БД (schema.prisma)
    
/frontend         - Модуль клиентского приложения
  /src
    /components       - Переиспользуемые Vue-компоненты
    /views            - Страницы сайта (Home, News, Prices и т.д.)
    /composables      - Функции-утилиты (useApi)
  /public/images    - Вся статика
```

## 🚀 Установка и запуск

Поскольку проект состоит из двух независимых частей, вам потребуется запустить их параллельно (или использовать Docker в будущем).

### Шаг 1: Подготовка БД

Скачайте и запустите XAMPP или любой локальный сервер MySQL по порту `3306`.
Создайте файл `backend/.env` и укажите там данные для подключения:

```env
DATABASE_URL="mysql://root:@localhost:3306/ask_dosaaf"
JWT_SECRET="sky_super_secret"
PORT=3000
```

### Шаг 2: Запуск Бэкенда (API)

Откройте первый терминал:

```bash
cd backend
npm install
npx prisma generate
npm run dev
```

Ожидаемый результат: сервер запустится по адресу `http://localhost:3000`.

### Шаг 3: Запуск Фронтенда (Vue)

Откройте второй терминал:

```bash
cd frontend
npm install
npm run dev
```

Ожидаемый результат: локальный Vite-сервер запустится по адресу `http://localhost:5173`. Фронтенд уже настроен (`vite.config.js`) проксировать API-запросы на бэкенд.

## ✨ Ключевые изменения

- **Авторизация**: Админ-панель на PHP сохранена, но для новых API эндпоинтов `/api/admin/*` работает система защиты через JWT токены.
- **Модальные окна**: Все старые jQuery модальные окна, слайдеры и Lightbox'ы переписаны на нативный Vue 3.
- **Оптимизация**: Больше никаких перезагрузок страницы (`a href`)! Теперь используется `vue-router`, что превратило сайт в SPA (Single Page Application).
